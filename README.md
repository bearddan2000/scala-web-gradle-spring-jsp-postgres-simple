# scala-web-gradle-spring-jsp-postgres-simple

## Description
A jsp springboot scala gradle build,
that connects to postgres database.

## Tech stack
- springboot
  - web
- jsp
- gradle
  - 6.8.2
- bootstrap
- jquery
- dataTable

## Docker stack
- postgres
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
